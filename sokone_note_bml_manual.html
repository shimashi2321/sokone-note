SOKONE NOTE SETUP GUIDE

底値ノートに
1タップで価格を送る方法

Chromeのブックマークレットを2つ登録するだけ。
商品ページを開いて、ブックマークをタップするだけで底値ノートに転送されます。

所要時間 約3分・無料
事前準備： このページ内の「コピー」ボタンでURLをコピーできます。Chrome（PC / スマホどちらも可）で進めてください。
1
ブックマークマネージャーを開く
最初の1回だけ行う準備です
1

Chrome右上の ⋮（縦三点メニュー）をクリックします。

2

ブックマークとリスト → ブックマークマネージャー の順に進みます。

2
「底値ノートに追加」を登録
新しい商品をはじめて登録するときに使うブックマーク
1

ブックマークマネージャー内で ⋮ をクリックし、新しいブックマークを追加 を選びます。

2

名前欄に以下を入力します。

底値ノートに追加
3

URL欄に以下のコードを貼り付けます。

javascript: ブックマークレット 
javascript:(function(){ var SOKONE_URL = "https://shimashi2321.github.io/sokone-note/"; function clean(s){ if(!s) return ""; var m = String(s).replace(/[０-９]/g, function(c){ return String.fromCharCode(c.charCodeAt(0)-0xFEE0); }).match(/[\d,]+(\.\d+)?/); return m ? m[0].replace(/,/g,"") : ""; } function pick(selectors){ for(var i=0;i<selectors.length;i++){ var el = document.querySelector(selectors[i]); if(el){ var t = el.getAttribute("content") || el.textContent; if(t && t.trim()) return t.trim(); } } return ""; } var host = location.hostname; var name = "", priceRaw = "", site = ""; if(host.indexOf("amazon") !== -1){ site = "Amazon"; name = pick(["#productTitle", "[itemprop='name']", "title"]); priceRaw = pick([ "#corePrice_feature_div .a-price .a-offscreen", "#corePriceDisplay_desktop_feature_div .a-price .a-offscreen", ".a-price .a-offscreen", "#priceblock_ourprice", "#priceblock_dealprice", "[itemprop='price']" ]); } else if(host.indexOf("rakuten") !== -1){ site = "楽天"; name = pick(["span.item_name", "h1", "[itemprop='name']", "title"]); priceRaw = pick([ ".price2", ".important", "[itemprop='price']", ".price" ]); } else if(host.indexOf("askul") !== -1){ site = "アスクル"; name = pick(["h1", "[itemprop='name']", "title"]); priceRaw = pick([".price", "[itemprop='price']"]); } else { site = host; name = pick(["[itemprop='name']", "meta[property='og:title']", "h1", "title"]); priceRaw = pick(["[itemprop='price']", "meta[property='product:price:amount']", ".price"]); } var price = clean(priceRaw); if(!name && !price){ alert("価格・商品名を取得できませんでした。ページの構造が対応外かもしれません。"); return; } var url = SOKONE_URL + "?name=" + encodeURIComponent(name) + "&price=" + encodeURIComponent(price) + "&site=" + encodeURIComponent(site) + "&siteUrl=" + encodeURIComponent(location.href); window.open(url, "_blank"); })();
4

保存 をクリックして完了です。

3
「底値ノートの既存アイテムに追加」を登録
同じ商品の新しい価格を、既存アイテムに追記するときに使うブックマーク
1

もう一度 ⋮ → 新しいブックマークを追加 を選びます。

2

名前欄に以下を入力します。

底値ノートの既存アイテムに追加
3

URL欄に以下のコードを貼り付けます。

使い分けの目安
底値ノートに追加

はじめて見る商品を、新しいアイテムとして登録したいとき。

既存アイテムに追加

すでに登録済みの商品で、別の日・別のお店の価格を追記したいとき。

商品ページを開いた状態でブックマークをタップすると、商品名・価格・サイト名が自動で底値ノートに送られます。
